Statify 📊

Statify is a simple web-based statistical analysis tool built using Django.

It allows users to upload datasets, perform basic statistical analysis, and visualize results through a clean and interactive dashboard.

---

🚀 Features

- Upload datasets (CSV / Excel)
- Preview data inside the browser
- Automatic statistical analysis:
  - Mean
  - Maximum
  - Minimum
  - Total records
- Dynamic data visualization using charts
- Simple insight generation based on data distribution
- Clean and modern UI design

---

🧠 Project Idea

Statify focuses on simplifying statistical analysis.

Instead of using complex tools, this project provides a lightweight and easy-to-use interface for analyzing real-world datasets, especially in:

- Medical data
- Engineering data

---

 Technologies Used

- Python
- Django
- Pandas
- Plotly
- HTML / CSS / Bootstrap
